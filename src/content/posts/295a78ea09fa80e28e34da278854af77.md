## 結論

- SSMS 匯入/匯出功能是靠 SSIS，沒裝就會直接鎖死
## 適合用在哪裡

- SSMS 出現「匯入/匯出資料」反灰
- 點擊時提示版本不支援
- 新環境或重灌 SSMS 後功能消失
## 流程步驟

### 1. 確認問題現象

- 在 SSMS 中對資料庫右鍵 → 匯入資料 / 匯出資料
- 發現按鈕反灰或顯示不支援
- 這不是權限問題，而是工具缺失（常見誤判）
### 2. 安裝 SSIS（核心解法）

- 安裝 SQL Server Integration Services（SSIS）
- SSMS 本身只是 UI，實際執行是 SSIS 在跑
- 沒裝 SSIS，就像有按鈕但沒引擎
### 3. 驗證功能恢復

- 重開 SSMS
- 再次右鍵 DB → 匯入/匯出資料
- 若可點擊，代表環境已完整
## 補充

- SSMS 與 SSIS 是分開安裝的（很多人會以為內建）
- 安裝 SQL Server 時，需勾選 Integration Services 才會有
- 有些精簡版本（例如 Developer Tools Only）不會包含 SSIS
## 收尾

- 這問題不難，但很容易卡住半天，少裝就是少靈魂
- 新環境記得先確認 SSIS，有些坑踩一次就好
