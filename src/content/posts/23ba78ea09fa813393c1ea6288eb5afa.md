## 結論

- 用 oh-my-posh + 主題設定，一行指令直接讓 PowerShell 變好看又好用
## 適合用在哪裡

- 想提升 CLI 使用體驗的工程師
- 長時間使用 PowerShell / Terminal 的人
- 想快速辨識 Git / 路徑 / 狀態資訊
## 流程步驟

### 1. 安裝 oh-my-posh

- 安裝工具（略過已安裝）
- 確認指令可用
```bash
oh-my-posh --version
```

### 2. 查詢主題

- 查看內建主題清單
```powershell
Get-PoshThemes
```

- 用來挑喜歡的樣式（建議先用預設測試）
### 3. 套用主題

- 初始化 PowerShell + 指定主題
```powershell
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\\jandedobbeleer.omp.json" | Invoke-Expression
```

- 會立即套用樣式（當前 session）
### 4. 設定永久生效

- 開啟 Profile
```powershell
notepad $PROFILE
```

- 加入以下內容
```powershell
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\\jandedobbeleer.omp.json" | Invoke-Expression
```

## 補充

- 字體需 Nerd Font（否則圖示會亂碼）
- Profile 找不到 → 先執行 `$PROFILE` 建立
- 主題可自由換（json 檔直接改）
## 指令 / 範例整理

<details>
<summary>Click to expend</summary>

```powershell
# 查詢主題
Get-PoshThemes

# 套用主題
oh-my-posh init pwsh --config "$env:POSH_THEMES_PATH\\jandedobbeleer.omp.json" | Invoke-Expression

# 編輯 Profile
notepad $PROFILE
```

</details>
